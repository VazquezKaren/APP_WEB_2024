Descripcion del proyecto

El proyecto propuesto utiliza el videojuego "Minecraft" como herramienta para fomentar un ambiente de convivencia sana en el salón de clases. Dado su acceso desde cualquier lugar, se designó a un administrador que creó un servidor en el juego. El código de invitación fue compartido a través de un grupo de WhatsApp; sin embargo, este código se difundió a otros grupos, lo que permitió que personas ajenas se unieran, dificultando el control y gestión de los usuarios autorizados por el administrador. Por esta razón, se decidió crear un gestor para este videojuego, el cual almacena información de cada servidor creado, como los administradores, códigos, jugadores, nombres y usuarios.
